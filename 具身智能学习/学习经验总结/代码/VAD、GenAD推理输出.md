# VAD、GenAD推理输出

VAD、GenAD 推理输出结构：

![Image](src/推理输出.png)

## 整体结构

```Plain
sample_token          → 样本唯一标识（对应 nuScenes 数据集的一帧）
├── detection_results → 3D 目标检测结果
├── map_results       → 向量地图（车道线等）
└── plan_results      → 自车规划轨迹
```

## 1\. `detection_results` — 3D 目标检测

| 字段            | 含义                   |
| --------------- | ---------------------- |
| `num_objects` | 当前帧检测到的目标数量 |
| `objects`     | 每个目标的详细信息数组 |

**每个 object 包含：**

| 字段                | 类型             | 说明                                                          |
| ------------------- | ---------------- | ------------------------------------------------------------- |
| `translation`     | `[x, y, z]`    | 目标中心点在 ego 坐标系下的位置（米）                         |
| `size`            | `[w, l, h]`    | 目标的宽度、长度、高度（米）                                  |
| `rotation`        | `[w, x, y, z]` | 四元数，表示目标的朝向角度                                    |
| `velocity`        | `[vx, vy]`     | 目标的速度（米/秒）                                           |
| `fut_traj`        | 数组             | **未来轨迹预测** — 模型预测该目标接下来 6 秒的运动轨迹 |
| `detection_name`  | 字符串           | 类别：`car`, `truck`, `pedestrian`, `barrier` 等      |
| `detection_score` | 浮点数           | 置信度分数（0\~1，越高越可信）                                |
| `ego_translation` | `[x, y, z]`    | 自车位置（通常`[0,0,0]`）                                   |

## 2\. `map_results` — 向量地图

| 字段            | 含义                             |
| --------------- | -------------------------------- |
| `num_vectors` | 向量地图元素总数（这里是 50 条） |
| `vectors`     | 每条地图元素的点序列             |

**每条 vector 包含：**

| 字段                 | 说明                                                                                     |
| -------------------- | ---------------------------------------------------------------------------------------- |
| `pts`              | 点序列`[[x1,y1], [x2,y2], ...]`，构成一条线                                            |
| `type`             | `0`=divider（车道分隔线），`1`=ped\_crossing（人行横道），`2`=boundary（道路边界） |
| `confidence_level` | 置信度，低于 0\.6 的通常会被过滤掉                                                       |

## 3\. `plan_results` — 自车规划轨迹

| 字段                      | 含义                                        |
| ------------------------- | ------------------------------------------- |
| `trajectories`          | **3 条候选轨迹**（对应 3 种驾驶命令） |
| `command_probabilities` | 模型对每种命令的预测概率                    |

**每条 trajectory 包含：**

| 字段             | 说明                                          |
| ---------------- | --------------------------------------------- |
| `command_id`   | `0`=右转, `1`=左转, `2`=直行            |
| `command_name` | 人类可读的命令名称                            |
| `trajectory`   | 轨迹点序列，每个点是`[dx, dy]` 的相对偏移量 |
